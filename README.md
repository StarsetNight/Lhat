## 介绍 INTRODUCE  
欢迎使用Lhat，这是一个基于PySide6的简易聊天程序。  
安全、简约、实用，这是我们的开发理念。
> 欢迎使用这款软件！  
> 要使用这款软件，下载它！  
软件分为两部分：客户端和服务端。  
- 客户端：一个“简单”的聊天程序。  
- 服务端：一个简单的聊天服务器。
